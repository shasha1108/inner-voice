## 入库规则（不可违反）

**入库条件（同时满足）：**
1. 内容已在小红书发布
2. 发布后数据表现好（用户反馈确认）
3. 用户明确指示"把这个入库"

**禁止入库：**
- 未发布的内容（包括文案已确认但未发布）
- 发布后数据差或普通
- 用户未明确要求入库
- agent 自行判断入库

**Why:** 资产库是爆款基因库，不是草稿箱。每一笔入库的内容必须是经过真实数据验证、用户确认有复用价值的内容。未经验证的内容入库 = 自嗨 = 污染库。

## 使用说明

**创作时查阅流程：**
1. 用情绪主题词匹配词条
2. 先查情绪类型字段（空间型/时间型外化）——决定是否适合图文媒介
3. 直接取用「推荐组合」或自选隐喻+角度

**词条格式说明：**
- **情绪类型**：空间型 / 时间型外化 / 时间型
- **选题来源**：用户输入 / 趋势推荐 / 数据迭代
- **叙事角度类型**：A感知细节 / B反向洞察 / C矛盾张力 / D功能归因
- **视觉策略**：ONE MASSIVE / 对比构图 / 信息层叠 / 可爱荒诞 / 毛边插画
- **发布数据**：点赞/收藏/评论/分享（用户提供）
