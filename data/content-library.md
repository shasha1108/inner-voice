### 安静的崩溃

**情绪类型**：空间型（外壳完整/内部碎裂）
**选题来源**：趋势推荐（趋势矩阵TOP 1 + 多源趋势交叉验证）
**本质定义**：不是摔碎在地上的崩溃，是所有碎片还悬在原来的位置——碎了但不倒

**核心隐喻**：「不会倒的废墟」
- 视觉主体：一座已经彻底碎裂的建筑，碎片悬浮在原位维持外形轮廓
- 对应读者身体经验：你靠在椅背上，感觉整个人随时会散架，但骨头还在原来的位置
- 物理形状：凝固/悬浮 — 碎了但没散
- 凝固不可能状态：建筑已碎但碎片违反重力悬浮原位

**叙事角度**：D功能归因型
- 归因角度：不倒不是意志力，是神经系统长期高压后启动的"功能维持模式"
- 重框方向：不倒不是坚强也不是麻木，是身体在用最后的方式保护你

**视觉策略**：ONE MASSIVE（碎裂建筑主体占60%+）+ 图3信息层叠（神经网络丝线连接碎片）
**搜索词命中**：情绪崩溃 / 深夜崩溃 / 假装没事 / 成年人崩溃

**金句库**：
| 金句 | 使用位置 |
|------|---------|
| 有一种崩溃，不会发出声音 | 命名前铺垫 |
| 「不会倒的废墟」 | 命名 |
| 不是你选择不倒。是你的身体，比你更早学会了一件事 | 归因 |
| 你碎了，但你不倒。你不倒，但你也不完整 | 矛盾 |
| 那些缝隙之间透出来的光，不是破碎的证据。是你还在这里的证据 | 重框结尾 |

**推荐组合**：隐喻「不会倒的废墟」+ D功能归因 + ONE MASSIVE视觉

### 安静的倦怠

**情绪类型**：空间型（在场但不在场）
**选题来源**：用户输入（06-08）
**本质定义**：不是崩溃，是大脑宕机——功能还在运转，但驾驶位是空的
**确认标题**：安静的倦怠｜你没有崩溃，你只是大脑宕机了

**核心隐喻**：「镜面期」（最终版正文不显式命名，意象自足）
- 视觉主体：鸟瞰大面积静止镜面水，极小人影在岸边，水面零涟漪
- 对应读者身体经验：你完成了今天所有的事，但你找不到自己在哪里
- 物理悖论：越平静越说明水下没有在流动——平静是症状，不是痊愈
- 凝固不可能状态：人在水边，水面无任何涟漪（人的存在应当制造波纹）

**叙事角度**：D功能归因型
- 归因角度：神经系统关掉情绪接收端，只维持最低社会功能（功能维持状态）
- 重框方向：平静不是好起来了，是给自己关机的允许——出口是回到身体里，不是振作

**注入概念**：功能维持状态（Functional Freeze）· psych-concepts.md 概念5

**视觉策略**：ONE MASSIVE（镜面水面占60%+，人影极小在边缘）
**搜索词命中**：安静的倦怠 / 精神内耗 / 职场倦怠 / 情绪耗竭 / 上班没有动力

**金句库（最终确认版）**：
| 金句 | 使用位置 |
|------|---------|
| 安静的坐下来，打开无数个文档 | 开场（量感词制造虚空落差；"安静的"呼应标题） |
| 那个想继续做事情的人，还是不是你 | 召唤（意义感消失，比"正在做的人"更精准） |
| 你还在工作，但完全不知道自己为什么工作 | 解构排比第二句（意义层丢失，比记忆层更深） |
| 你还可以笑，但笑完却不明白自己为什么要笑 | 解构排比第三句（情绪顶点，理解层丢失） |
| 不是你变麻木了，而是你的身体，一直在替你强撑着，但只是撑着 | 归因（麻木比冷漠精准；"撑着/撑着"重复锚点） |
| 水面越平静，越说明——水下，已经没有什么东西在流动了 | 归因意象（无命名，意象自足；只用一次） |
| 把自己的双脚用力踩在地面上，去感受那种踏实的感觉 | 重框行动锚点（主动动词有身体感，比"注意感觉"更能拉回当下） |
| 这里很安全，放轻松的休息一下吧 | 结尾（陪伴感温度，与低温全篇基调一致） |

**创作规律（两轮改稿提炼）**：
- 命名可以不揭晓：意象足够自足时，不显式命名反而留白感更强（参见铁律12）
- 排比三句要升意义层次：记忆丢失→意义丢失→理解丢失，不只是换场景（铁律10）
- 标题词渗入开场动作："安静的坐下来"呼应"安静的倦怠"（铁律8）
- 量感词制造落差："无数个文档" vs 光标空闪（铁律9）
- 麻木比冷漠精准：神经层面的关闭用麻木，态度层面的关闭用冷漠（铁律11相关）
- 行动动词有身体感："用力踩"比"注意感觉"更能把人拉回当下（铁律13）
- 结尾温度与全篇一致：低温轻柔→陪伴感结尾（铁律14）

**推荐组合**：隐喻「镜面期」+ D功能归因 + ONE MASSIVE（鸟瞰镜面水）

### 被做成了产品的孤独

**情绪类型**：空间型（人与人之间的空隙被商品化）
**选题来源**：趋势推荐（06-15 机器人伴侣热点 + 多源趋势交叉验证）
**本质定义**：不是没有人——是人与人之间的距离已经大到可以装下一个产业。孤独不再是私密的体验，而是被明码标价的商品。
**确认标题**：我们的孤独，终于被做成了产品。

**核心隐喻**：「让出来的位置」（非「」显式命名——概念通过标题和正文充分展开，不依赖「」即可被读者复述）
- 视觉主体方向：两个人/人群之间的巨大空隙，或一个空位周围站满了看手机的人
- 对应读者身体经验：你在人群中，每个人都在看手机，没有人抬头——你和一个机器人之间的距离比和任何人都近
- 物理悖论：在"连接最多"的时代里，人与人之间反而空出了最大的距离

**叙事角度**：B反向洞察型
- 反常识开场：机器人伴侣不是AI的胜利，是人与人距离的体温计
- 重框方向：机器人填补的位置，恰好提醒我们那个位置是用来放彼此的

**视觉策略**：待定（文案先确认，视觉后续讨论）
**搜索词命中**：机器人伴侣 / 孤独 / 人与人之间的距离

**金句库**：
| 金句 | 使用位置 |
|------|---------|
| 我们的孤独，终于被做成了产品 | 标题/开篇 |
| 人与人之间的距离，已经大到可以装下一个产业了吗 | 召唤/钩子 |
| 不是因为算法和代码终于懂得了爱，而是人与人之间的防备与距离，已经大到了一种我们不愿承认的程度 | 矛盾/归因 |
| 我们已经不想再花费更多的精力和时间去认识一个人、了解一个人，甚至爱上一个人了 | 痛点/追问 |
| 机器人填补的，是一个让我们能够彼此倾听和关爱的位置 | 重框/结尾 |

**创作规律（双版对比提炼）**：
- 标题即第一句：零浪费开篇，不预告
- 敢说读者不愿承认的："不想再花精力去认识一个人"——追问"为什么没力气"，不只描述"没力气"
- 结尾往前看："填补的是让我们重新看见彼此的位置" > "那个位置原本是留给彼此的"
- 口语直觉："没完没了""多余的力气" > 书面语替代
- 无结构痕迹，信任读者：不命名不需要命名的概念，不硬塞CTA
- B-HF3/B-HF4均未触发（概念清晰不需要「」；社会热议话题不需要CTA）

**推荐组合**：隐喻「让出来的位置」（非显式命名）+ B反向洞察

### 深度感受但不破碎（高敏感的承受力悖论）

**情绪类型**：空间型（感知通道宽/排出通道窄——进水口大两倍，排水口和所有人一样粗）
**选题来源**：全球趋势交叉验证（Reddit r/emotionalintelligence "How to feel deeply without falling apart" 108↑ + r/Adulting 5649↑ + 小红书多词验证）
**本质定义**：深度感受的能力正在变成负担——读者需要的不是变钝，是保持锋利但不割伤自己
**确认标题**：太敏感的人 ｜ 致那个被说过这句话的你

**核心隐喻**：发光薄壳（非「」显式命名——正文用"进水口/排水口/感知系统/过滤层"的物理机制承载，意象通过视觉传达）
- 视觉主体：一个巨大、半透明、发光的薄壳体悬浮在深色空间，冷色粒子从外部穿过壳壁变成暖金色
- 对应读者身体经验：同事的叹气穿过工位落在胸口；一句话在脑子里循环三天
- 物理悖论：薄不是弱点——正因为薄，光才能进来。壳不挡住任何东西，只让穿过它的东西变了温度
- 配色方向：深紫底色 (#351E28) + 暖橙色微光渗透，冷暖对比

**叙事角度**：D功能归因型
- 归因角度：感知系统天生没有"不重要"的过滤层——所有刺激同等强度抵达。进水口比别人大两倍，排水口和所有人一样粗
- 重框方向：涌进来的不等于要留在里面。水不是库存，水是流。你只是管道比别人宽——要学的不是怎么变窄，是怎么让进来的流出去

**视觉策略**：ONE MASSIVE（发光薄壳占画面60%+）
**搜索词命中**：太敏感的人 / 高敏感 / 为什么别人没事 / 想太多 / 容易共情

**金句库**：
| 金句 | 使用位置 |
|------|---------|
| 同一场雨，落在别人那里只是地面的潮湿。落到你这里，就是一条河。 | 命名/展开 |
| 你关掉的不只是痛苦。 | 情感转折 |
| 你需要的从来不是少感受一点。你只是需要一个地方——把不属于你的重量，轻轻放下来。 | 重框结尾 |

**创作规律（本轮提炼）**：
- 不造概念、不解释概念——用"进水口/排水口"的物理事实代替「」命名，读者不需要学一个新词
- 场景颗粒度决定共鸣深度——"同事的叹气+脑内3秒独白"比通用场景更有锚定力
- 视觉和文案用不同的载体表达同一机制——文案用管道/水流，视觉用发光薄壳/冷→暖转化，互不绑定但指向同一件事

**推荐组合**：机制归因（进水口/排水口）+ D功能归因 + ONE MASSIVE（发光薄壳）
