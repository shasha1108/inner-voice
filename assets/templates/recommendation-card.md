# 选题推荐卡片模板

> 用于 STEP 0.3 输出选题推荐。

## 输出格式（人类可读）

```
【选题推荐】（N个方向）

方向1：[情绪方向]
核心悖论：[待命名的感受——"看似A，实则B"格式]
切入角度：[具体角度]
信号来源：[最先出现的平台 + 热度数据]
趋势温度：[轻量幽默 / 温暖治愈 / 严肃沉重 / 冷静克制]
视觉锚点：[能让人3秒识别这个热点的具体视觉元素]
传播因子：[用户转发原因——好笑/被戳中/可爱/被理解]
Pinterest调研：[配色方向 + 构图特征 + 小红书尚无的差异化机会]
搜索价值：[高(1w+) / 中(1k-1w) / 低(<1k)] — [验证依据]
全球信号强度：[强/中/弱] — [Reddit/X数据]
简中竞争烈度：[红海/中等/蓝海] — [小红书前10条分析]
写作难度：[高/中/低] — [判断依据]
视觉差异化：[具体的差异化视觉机会]
推荐指数：[1-5星] — [综合判断]

方向2/3：（同格式）
```

## 飞书导出格式（STEP 6.5 选【是】时使用 · 每个方向组装一个 JSON）

```json
{
 "情绪主题词": "[emotion_theme]",
 "选题": "[方向标题，如：安静的崩溃｜你没有崩溃，只是悄悄把自己调成了静音]",
 "核心悖论（待命名的感受）": "[看似A，实则B的一句话]",
 "信号来源追溯": "[平台 + 热度，如：Reddit r/mentalhealth 138↑ + X话题上升中]",
 "趋势温度": "[轻量幽默 / 温暖治愈 / 严肃沉重 / 冷静克制]",
 "视觉锚点": "[具体视觉元素]",
 "传播因子": "[用户转发原因]",
 "Pinterest 视觉调研": "[配色方向；构图特征；差异化机会，三项用分号分隔]",
 "搜索价值": "[高(1w+) / 中(1k-1w) / 低(<1k)]",
 "全球信号强度": "[强 / 中 / 弱]",
 "简中竞争烈度": "[红海 / 中等 / 蓝海]",
 "写作难度": "[高 / 中 / 低]",
 "视觉差异化": "[具体差异化机会描述]",
 "推荐指数": [1到5的数字],
 "添加时间": "[YYYY-MM-DDThh:mm:ss+08:00]"
}
```

## 要求

- 每个方向必须基于实际搜索验证（OpenCLI 多源趋势 + 小红书搜索验证）
- 竞争程度基于小红书前10条内容质量判断
- 检查 是否已做过相似选题（对照 data/content-library.md）
- 飞书导出 JSON 中所有字段必须填写，不留空字符串
