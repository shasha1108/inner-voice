# 叙事角度卡片模板

> 用于 STEP 4 人工确认点 A-2 输出。基于已确认的 `selected_metaphor` 展开。

## 输出格式

```
【角度 A · 感知细节入场型】
一句话概括：｜触达情绪：｜核心隐喻：（来自selected_metaphor）｜视觉可行性：低/中/高｜钩子预判：｜搜索词植入建议：[该角度能自然植入哪些搜索热词]

【角度 B · 反向洞察型】
（同上格式）

【角度 C · 矛盾张力型】
（同上格式）

【角度 D · 功能归因型（新增）】
一句话概括：｜触达情绪：｜核心隐喻：（来自selected_metaphor）｜归因角度：[为什么会这样的一句话解释]｜重框方向：[换个角度看的新理解]｜视觉可行性：低/中/高｜钩子预判：｜搜索词植入建议：
```

## 要求

- 四个角度必须有本质差异
- 核心隐喻必须来自 STEP 3 的候选列表，禁止临时生成新隐喻
- 核心隐喻必须有人物、有场景、有故事感，而非干燥物象

## 角度D说明

- "归因"不是科普，用同行者语气说出"为什么会这样"
- "重框"不是行动建议，是"换个角度看这件事"
- 是否在正文中展开归因段落，取决于内容类型和角度适配性（见 STEP 3）
