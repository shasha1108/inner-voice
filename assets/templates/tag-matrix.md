# 标签矩阵输出模板

> 用于 STEP 7 标签生成输出。

## 输出格式

```
【标签矩阵】（15个标签，按搜索量分层）
▎搜索入口词（2个，月搜1w+）：#xxx #xxx
▎垂类精准词（6个，月搜1k-1w）：#xxx #xxx #xxx #xxx #xxx #xxx
▎长尾场景词（5个，月搜100-1k）：#xxx #xxx #xxx #xxx #xxx
▎形式标签（2个）：#xxx #xxx

【一键复制版】
#xxx #xxx ... （全部）
```

## 生成步骤

1. **搜索量验证**（联网查询小红书搜索建议词热度，或基于经验评估）
2. **竞争度评估**（搜索该词，前10条内容质量如何）
3. **组合输出**

## 标签分层规则

| 层级 | 月搜索量 | 数量 | 作用 |
|------|---------|------|------|
| 搜索入口词 | 1w+ | 2个 | 吃搜索流量 |
| 垂类精准词 | 1k-1w | 6个 | 精准匹配目标用户 |
| 长尾场景词 | 100-1k | 5个 | 从文案提炼的具体场景 |
| 形式标签 | — | 2个 | 内容形式标识 |

## 禁止

- 不得堆砌「情感」「成长」「自我」等无差异超泛词
- 「治愈」不单独用 #治愈
- 形式标签必须与实际配图风格一致（#情绪插画 = 插画风；#情绪图文 = 图文组合）
