## 核心原则

1. **双轨独立评分。** 标题和正文各走各的维度，不混为一谈。
2. **每个维度打满分为优秀，不是"不扣分"。** 一个维度满分 = 这个维度做到了最好。
3. **评分必须诚实。** 如果拿了满分但实际不行，脚本会暴露出来。
4. **脚本计算结果为准。** 人工判断用于填分值，但总分/通过/不通过由脚本输出。

## 硬性不及格条件（触发任一条 → 对应轨总分上限 3.0）

以下条件触发后，该轨（标题/正文）直接判不及格。

### 标题硬性不及格

| # | 条件 | 为什么致命 |
|---|------|-----------|
| T-HF1 | 声称搜索友好但无小红书验证数据（H1=0） | 自造 SEO 数据，创作者被欺骗 |
| T-HF2 | 竖线前后是断裂的两个独立句子（H3=0） | 标题破碎，读者无法理解 |
| T-HF3 | 使用 «» 书名号做标签装饰（H3=0） | 过时手法，暴露 AI 感 |

### 正文硬性不及格

| # | 条件 | 为什么致命 |
|---|------|-----------|
| B-HF1 | 出现绝对禁用句式（B7=0） | 账号视角崩坏，读者取关 |
| B-HF2 | 隐喻用法混用——前半段当概念、后半段当物理空间（B3=0） | 隐喻崩溃，读者困惑 |
| B-HF3 | 「」命名缺失，或凭空出现无铺垫无引出（B3=0） | 没有给读者的感受一个新名字 = 看完就忘。**例外：** 当全文核心洞察本身即为对已知事物的重新命名（标题已完成了"命名"动作——如"我们的孤独被做成了产品"），且正文通过排比/递进/追问将该洞察充分展开时，「」非必须。判断标准：删掉「」后，读者能否准确复述这篇"说了一件什么事"？能 → B-HF3 不触发 |
| B-HF4 | 无 CTA（B8=0） | 读者不知道怎么互动 → 没有评论/收藏/转发。**例外：** 当内容涉及社会热议话题（有天然讨论属性），且结尾已给出完整观点、能自然引发读者"想说点什么"时，CTA 非必须。判断标准：读完最后一句，你是否感觉读者会有想讨论的冲动？能 → B-HF4 不触发 |
| B-HF5 | 内容为教程/素材/壁纸类（B6=0） | 这不是本账号的价值 |

## 标题评分维度（满分 10 分 · 通过线 ≥ 9.0）

```
python3 scripts/score.py --mode title --h1 X --h2 X --h3 X --h4 X --json
```

### H1 · 搜索可见性（0-2.5）

> **核心问题：标题里有能被搜索到的词吗？这个词是句子自然的一部分吗？**

| 分数 | 标准 |
|------|------|
| 0 | 声称X是长尾搜索词但无任何小红书搜索结果支撑 → 硬性不及格 T-HF1 |
| 0.5-1.0 | 标题不含任何可搜索词，或搜索词是硬塞进去的（读起来不自然） |
| 1.1-1.8 | 有 1 个可搜索词，但搜索量低（<1k），或者词是贴上去的不是自然的 |
| 1.9-2.5 | 有 1+ 个可搜索场景词，且是句子自然的一部分——读者搜这个词能找到你 |

**关联铁律：** #1, #12, #13

---

### H2 · 隐喻关键词（0-2.5）

> **核心问题：标题是否包含本次确认的隐喻核心词？这个词在标题里是视觉锚点还是抽象概念？**

| 分数 | 标准 |
|------|------|
| 0-0.8 | 标题不含隐喻核心词，或隐喻词被抽象化（"边界感"替代了"圈"） |
| 0.9-1.6 | 包含隐喻词但不够突出，读者不一定注意到 |
| 1.7-2.5 | 隐喻核心词是标题的视觉锚点——读者看到标题就能联想到封面画面 |

**关联铁律：** #4, #5, #35

---

### H3 · 人话感（0-2.5）

> **核心问题：大声读出来——这是人在说话吗？**

| 分数 | 标准 |
|------|------|
| 0 | 竖线前后是断裂的两个独立句子 → 硬性不及格 T-HF2 |
| 0 | 使用 «» 书名号做标签装饰 → 硬性不及格 T-HF3 |
| 0.1-1.0 | AI 翻译感重，读起来不像中文直觉输出 |
| 1.1-1.8 | 基本通顺，但有一两处不像人话（如修饰关系不成立："好话。就是太亮了。"——亮不修饰话） |
| 1.9-2.5 | 大声读出来就是人在说话——标题不是优化出来的，是说出来然后写下来的 |

**关联铁律：** #2, #6, #7, #9, #10, #11

---

### H4 · 钩子力（0-2.5）

> **核心问题：读者在信息流里看到这个标题，0.5 秒内会被钩住吗？**

| 分数 | 标准 |
|------|------|
| 0-0.8 | 泛化声明（"我们常常""每个人都会"），或纯粹情绪标签，没有具体画面 |
| 0.9-1.6 | 有具体概念但不够意外——"是的，然后呢？"（读者不会想点开） |
| 1.7-2.5 | 有反常识/反转/具象场景/复合钩子——读者被钩住了，必须点进来看 |

**关联铁律：** #3, #8, #12

---

**🚨 人话优先原则（H1/H2 下限保护）：** 标题的第一优先级是让读者觉得「这是人在说话」。当 H3（人话感）≥ 2.3 时，H1 和 H2 的分数下限提至 1.5——人话好的标题，搜索弱一些也不应该被罚到及格线以下。

**🚨 人话满分豁免：** 当 H3=2.5 且 H4≥1.5 时，标题视为通过（总分 ≥9.0），不论 H1/H2 的实际得分。理由：一个读起来完美像人说话的标题本身就是最高质量的钩子——读者先被这句话停下来，搜索才有意义。不要把一个人话满分的标题强行塞进搜索词。

## 正文评分维度（满分 10 分 · 通过线 ≥ 9.5）

```
python3 scripts/score.py --mode body --b1 X --b2 X --b3 X --b4 X --b5 X --b6 X --b7 X --b8 X --json
```

### B1 · 角度执行（0-1.5）

> **核心问题：这篇文案有没有把观点讲清楚？结构是服务于观点的，不是反过来。**

角度模板（A感知/B反向/C矛盾/D归因）是用来帮助组织思路的框架，不是评分必须穿上的衣服。如果一篇文案有自己的有机结构，并且观点被清晰地推进和讲清楚了，它不需要严丝合缝匹配一个模板。

| 分数 | 标准 |
|------|------|
| 0-0.5 | 结构混乱——段落之间没有逻辑推进，读者读完不知道作者想说什么 |
| 0.6-1.0 | 观点能识别但推进不够清晰——方向对，但中间有跳跃或不够充分 |
| 1.1-1.5 | 观点被清晰地推进和讲清楚：开头确立了方向、中间有实质推进、结尾收住了——不管是否匹配角度模板，有机结构同样满分 |

> ⚠️ 如果文案有自己的有机结构且讲清楚了观点，即使不完全匹配 A-2 确认的角度模板，B1 不低于 1.0。结构是脚手架，不是模具。

**关联铁律：** #41, #42

### B2 · 开头钩子（0-1.5）

> **核心问题：开头是否属于 10 种钩子类型之一？是否能在 3 秒内让读者停下来？**

| 分数 | 标准 |
|------|------|
| 0-0.5 | 开头不属于任何一种钩子类型——泛化声明或情绪标签开场 |
| 0.6-1.0 | 属于某一种钩子类型，但执行偏弱——如反问式但问题太普通 |
| 1.1-1.5 | 钩子类型明确 + 即刻具象（物理时刻/具体动作/反常识陈述），3 秒停滑 |

**关联铁律：** #15, #30, #49

### B3 · 隐喻一致性（0-1.5）

> **核心问题：同一个隐喻在全文中使用方式一致吗？动词和宾语匹配吗？「」命名有铺垫吗？**

| 分数 | 标准 |
|------|------|
| 0 | 隐喻前半段当概念用、后半段当物理空间用（混用）→ 硬性不及格 B-HF2 |
| 0 | 「」缺失 + B-HF3 触发（概念不清晰 + 读者无法复述核心洞察）→ 硬性不及格 B-HF3 |
| 0.3-0.5 | 「」缺失但 B-HF3 未触发——概念通过标题和正文已充分展开，读者能复述核心洞察。不扣分起点。 |
| 0.6-1.0 | 「」存在但引出方式偏弱；或隐喻基本一致但有一处动词/宾语微瑕 |
| 1.1-1.5 | 全篇隐喻使用方式一致 + 动词宾语全部匹配 + 「」（如有）引出自然有铺垫；或无「」但底层意象贯穿全文且无任何混用 |

**关联铁律：** #35, #36, #37, #38, #39, #40

### B4 · 矛盾句质量（0-1.0）

> **核心问题：矛盾句是从思考中自然长出来的，还是被"还差一组"逼出来的？**

矛盾句是一篇文案自然产生的副产物，不是一个必须凑够的 KPI。不设数量门槛——一篇文案可以有 1 组极其有效的矛盾句，也可以有 5 组自然涌现的。关键是：每组矛盾句删掉后，文章的逻辑会断吗？会 → 有效；不会 → 那组不算。

| 分数 | 标准 |
|------|------|
| 0-0.3 | 没有有效矛盾句，或矛盾句明显是凑数堆砌——删掉后前后文完全不受影响 |
| 0.4-0.7 | 有 ≥1 组有效矛盾句（从隐喻中自然产生、删掉会导致逻辑断裂）但深度不够或类型单一 |
| 0.8-1.0 | 有 ≥1 组有效矛盾句，且具有真正的悖论张力——不是"A↔B"的格式，而是"两种对立状态同时为真"的洞见 |

**关联铁律：** #20, #43, #46, #47

---

### B5 · 情绪弧线（0-1.0）

> **核心问题：文案有情绪上升弧线吗？还是从头平到尾？**

**前置判断：这篇的情绪基调是「需要被唤醒」还是「需要被陪伴」？**

- 需要被唤醒（如身份撕裂、自我怀疑、愤怒被压抑）→ 需要上升弧线
- 需要被陪伴（题材本身是安静的理解和接纳）→ 弧线平缓是合理的——推进方式是「困惑→轻轻明白」，不是「被压住→愤怒」

| 分数 | 标准 |
|------|------|
| 0-0.3 | 没有推进——读完和没读一样的感受 |
| 0.4-0.7 | 有推进但中间有断层——某个段落情绪跳跃了 |
| 0.8-1.0 | 有清晰的情绪推进——不管是上升弧线（被压住→愤怒→力量）还是平缓弧线（困惑→理解→释然），读者读完的感受比读之前完整了一点 |

**关联铁律：** #16, #17, #48, #50

### B6 · 价值增量（0-0.75）

> **核心问题：读者读完中间部分获得了什么？知识/情绪/审美/社交价值有至少一种吗？**

| 分数 | 标准 |
|------|------|
| 0 | 内容是教程/素材/壁纸类 → 硬性不及格 B-HF5 |
| 0.1-0.25 | 读者读完回答"获得了什么？"→ 答不上来 |
| 0.3-0.5 | 有某一种价值但不够充分（"有点被说中但不够击中"） |
| 0.6-0.75 | 提供了明确的价值——读者能说出"这篇让我知道了X"或"这篇让我感到Y" |

**关联铁律：** #31, #51

### B7 · 语言质感（0-1.5）

> **核心问题：每句话读起来是中文直觉写成，还是英文思维翻译的？**

| 分数 | 标准 |
|------|------|
| 0 | 出现绝对禁用句式（不是X而是Y/你应该/学会/记住/请记得/很多人都/治愈自我标签/我们把这种感觉叫做/心理学上称之为/这其实是/那个负责X的你）→ 硬性不及格 B-HF1 |
| 0.1-0.5 | AI 腔明显，翻译感重，或有多处拗口；排版缺失（段落未空行、结尾粘在一起） |
| 0.6-1.0 | 通顺但平淡，缺少口语节奏，偶有翻译腔；排版基本规范但有一两处疏漏 |
| 1.1-1.5 | 全篇中文直觉写成，短句节奏好，零翻译腔，逐句朗读无拗口；排版完整：段落之间空行、命名句前后各空一行、结尾每句独立成行 |

**关联铁律：** #14, #19, #22, #23, #24, #25, #26, #27, #28, #29

### B8 · CTA 有效性（0-1.25）

> **核心问题：结尾有没有行动指令？这个 CTA 能触发 >2 个字的具体讨论吗？**
>
> **克制原则：** 如果结尾本身已经干净有力地收住了，CTA 不是必须的。沉浸式文案末尾硬接互动 = 刚建立的情绪空间被戳破。判断标准：读完全篇最后一句 → 直接读 CTA → 中间有没有情绪断层？有断层 → B8 上限 0.5。

| 分数 | 标准 |
|------|------|
| 0 | 无 CTA + B-HF4 触发（话题无天然讨论性 + 结尾未形成完整观点）→ 硬性不及格 B-HF4 |
| 0.3-0.5 | 无 CTA 但 B-HF4 未触发——结尾已给出完整观点，话题本身有讨论基础。不扣分起点。 |
| 0.6-0.9 | 有 CTA 但太弱——读者不知道要做什么，或 CTA 在解释自己的符号（如「⭕代表...」）；或 CTA 破坏了结尾的情绪完整性（硬接互动、情绪断层） |
| 1.0-1.25 | CTA 清晰有效——读者知道留什么；且 CTA 是内容自然延伸，读完全篇最后一句到 CTA 之间没有情绪断层 |

**关联铁律：** #18, #32, #33, #34, #44, #45

## 结构信号检查（评分前必须先过 · AI 识别「格式制造」的痕迹）

> AI 无法判断「这篇内容是否在认真思考」——因为 AI 自己生成文本的路径和「模拟思考」的路径是一样的。但 AI 能识别**格式制造的痕迹**：那些不是因为思考需要而出现、而是因为「一篇文案应该长这样」而出现的结构特征。
>
> 以下 5 个信号，逐条检查。**触发 ≥ 2 个 → 本内容有格式制造嫌疑 → 所有维度评分完成后，总分扣 1.0 分。信号 5 单独触发 → 评分体系导致的过度优化，不加分不扣分，但标记提醒。**

### 信号 1 · 结论可预测性

> **读完标题，能猜到这篇的结论吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 结论里有至少一处，在标题里完全猜不到——不是套路收尾，是真正想通了一个东西 |
| ❌ 触发 | 标题就是结论的压缩版——读完标题就知道结尾会说什么，中间只是填充 |

**检查方法：** 先读标题，猜一个结论。再读结尾。一样 → 触发。

---

### 信号 2 · 中间转折

> **正文中间有没有至少一处思考「拐弯」的地方？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有一处「但我后来发现...」「其实不是这样的...」「后来才想通...」——不是模板句式，是真的推进了一层 |
| ❌ 触发 | 每段严格对应一个结构功能（描述现象 → 揭示原因 → 给出新视角），没有多余的、意外的内容 |

**检查方法：** 逐段问「这一段如果删掉，前后文逻辑会断吗？」——如果每段都可以删而逻辑不断，说明段落之间没有思考推进，只是按模板排列。

---

### 信号 3 · 细节特异性

> **正文里有至少一个「只有这篇才会出现」的具体细节吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有一个时间/地点/动作/物件的组合过于具体，不像是为了凑场景编出来的 |
| ❌ 触发 | 场景全部是通用模板——换成另一个情绪主题也能直接用 |

**检查方法：** 把情绪主题词换掉（如「焦虑」换成「孤独」），场景还能直接用吗？能 → 触发。

---

### 信号 4 · 语言均匀度

> **全篇句子长度、节奏、精致度高度统一吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有粗糙的地方——某句话太短了、太长了、太口语了——但因为它是真实的，读起来不觉得突兀 |
| ❌ 触发 | 全篇像被抛光过——每句话差不多的长度、差不多的精致度、没有一句「出格」的 |

**检查方法：** 找全篇最短的一句和最长的句子。差距 < 3 倍字数的 → 语言过于均匀 → 触发。

---

### 信号 5 · 结构要素挤压感

> **正文的每一个段落是否都在「完成一个评分任务」？**

这是评分体系的副作用——agent 为了满足 B2（开头钩子）B5（情绪弧线）B4（矛盾句）而生产出结构完美的文案，但每个结构要素之间没有思考推进，只有任务切换。原文人话→优化后更像 AI：转折在、弧线在、矛盾句在，但离人话更远了。

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 段落之间的过渡是自然推进——「然后我开始想…」「后来才意识到…」——下一个想法是从上一个想法里长出来的，不是从评分表里勾出来的 |
| ❌ 触发 | 至少两处段落过渡有「齿轮切换」感——这段在完成「反常识开场」、那段在完成「愤怒转折」、下一段在完成「重新框架」——结构要素的边界清晰得像是不同的人写的 |

**检查方法（两问，任一答"是"→触发）：**

1. **功能标签测试**：给每段贴一个结构功能标签（开场钩子 / 痛点展开 / 愤怒转折 / 归因解释 / 重框 / CTA）。有没有两段以上的标签切换让读者感到「上一段还在说 X，怎么突然跳到 Y 了」？有 → 触发。

2. **删段测试**：删掉情绪转折段，前后文是断了吗？还是更顺了？如果删掉后读起来更自然 → 那个转折段是被评分逼出来的，不是思考需要 → 触发。

---

## 禁止行为

- ❌ 跳过结构信号检查直接评分（这 4 个信号必须在评分数之前过完）
- ❌ 不填分值直接调脚本（脚本需要数值，先判断再填入）
- ❌ 硬性不及格触发后给该维度打正分（触发=该维度=0）
- ❌ 预测只说好话（"这篇会火的"无任何依据 = 无效预测）
- ❌ 修改优先级不说具体维度（"优化一下"不够，必须说"先修 B7 语言质感"）
- ❌ 标题和正文混评（双轨独立，各走各的分值）
- ❌ 视觉不在本阶段评——视觉 Brief 质量在复盘环节用数据回测
