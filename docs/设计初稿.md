# 心球 XinQiu · 青少年像素情绪养成设计初稿

> 2026-06-26 · 脑暴产出
>
> 组合引擎：inner-voice（情绪命名）+ pixel-bloom（像素生命体）+ healing-space（触觉交互）

---

## 产品定位

一个像素情绪养成小程序。每天 3 分钟，青少年通过"触碰→命名→喂养→生长"四步循环，把看不见的感受变成一颗只属于自己的像素生命体。

**一句话：你的球什么都不说。但每天打开它，你就知道自己还活着、还在变、还在亮。**

---

## 核心理念

成年人的情绪产品："我帮你命名你的感受" → 被动、内省
青少年的情绪产品："你的感受长什么样子？" → 主动、创造、好玩

### 三条设计铁律

1. **不治疗，只见证。** 不上诊断标签，不说教，不推送"你今天心情不好可以试试这样做"。心球只是在那里，等你来喂。
2. **不暴露，只选择。** 不让青少年"说"出感受，让他们"选"一张卡、"起"一个名字。安全距离 = 表达意愿。
3. **不可预测，但可靠。** 形态变化不可预测（每天的期待），但每天固定时间等你（安全感）。断了一天球不会死，只会暗一点。

---

## 核心玩法

```
┌─────────────────────────────────────────┐
│                                         │
│  1.触碰 ──→ 2.命名 ──→ 3.喂养 ──→ 4.生长  │
│   (选卡)    (起名)    (喂给它)   (它变了)   │
│                                         │
└─────────────────────────────────────────┘
```

### Step 1：触碰 — "今天哪张卡像你？"

9 张像素情绪卡片。不是"焦虑""抑郁""开心"这种临床标签，是具象视觉 + 同伴语气：

| 卡面 | 文案 |
|------|------|
| 🌀 慢慢转的漩涡 | "今天好像被什么东西卷进去了" |
| 🫧 透明气泡 | "周围很吵，但我听不太清" |
| 🌑 不亮的星 | "没发生什么，就是亮不起来" |
| ⚡ 快断的线 | "一直在绷着，不知道能撑多久" |
| 🌱 石缝里的芽 | "其实今天有一件很小很小的好事" |
| 🪞 碎了但没散的镜子 | "今天很分裂，但居然撑住了" |
| 🌊 退潮后的沙滩 | "热闹过去了，剩我一个人" |
| 🕳️ 安静的洞 | "不想说话，也不想被人找到" |
| 🔥 小火苗 | "今天有点生气，但不知道为什么" |

**技术：** inner-voice STEP 0 每日趋势感知 → 动态更新卡片库。今天全球青少年在焦虑什么，卡片就出现什么。不是静态题库，是活的。

### Step 2：命名 — "你想叫它什么？"

给选中的感受起名字。三种方式：

- 一句话：自由输入
- 一个代号：AI 给 3 个建议（如"透明人""泡泡日""安静模式"）
- 或什么都不写，就叫"今天的那个东西"

**技术：** inner-voice STEP 3 隐喻命名引擎简化版——不做深度悖论分析，做"帮你说出你想要的词"。

**关键设计：** 自己命名 = 掌控感 = 心理赋能。14 岁的孩子最讨厌"被定义"。

### Step 3：喂养 — "喂给你的心球"

你养了一颗像素"心球"。它每天等你来喂。

选好卡 + 起好名 → 变成一种颜色的光点 → 按住拖过去 → 松手 → 它吞下去。

**视觉反馈：**
- 吃下"漩涡" → 表面泛起紫色涟漪
- 吃下"气泡" → 心球半透明，微微漂浮
- 吃下"发芽" → 长出小片像素叶子
- 吃下"碎镜" → 表面浮现裂纹，但核心发光

**技术：** pixel-bloom Canvas 渲染引擎 + healing-space 触觉交互（拖拽手感要有"重量"）。

### Step 4：生长 — "7 天后，它长成了你的样子"

同一个生命体，连续喂养后不断变化。变化不可预测——不是"高兴就长叶子、难过就枯萎"的死板映射，而是**情绪组合产生意想不到的形态**。

每天的情绪与前几天积累加权混合，每个心球独一无二。你的焦虑 + 第二天的平静 + 第三天的愤怒 → 心球长出尖刺但核心发光。别人同样三天 → 完全不同的物种。

**技术：** pixel-bloom 程序化生长算法 + inner-voice 情绪向量 → 形态参数映射。非线性、非一对一。

## 习惯培养

| 机制 | 设计 |
|------|------|
| 连续喂养 | 断一天球不会死，但会"暗淡"。重新喂了恢复。不是惩罚，是牵挂 |
| 7 天解锁 | 每周解锁新场景：池塘 → 森林 → 星空 → 深海 |
| 隐藏成就 | 30 天连续喂养 → 心球"醒来"——开始偶有自主动画，不是你拖拽才动 |
| 无推送 | 不推送"你今天还没喂养哦"。心球靠牵挂，不靠通知 |

## 心理疗愈

### 1. 匿名共鸣："有人和你的球很像"

系统匿名匹配形态高度相似的心球。你可以"送一颗像素花"，或者什么都不做。看不到对方身份。

不是社交功能。是"你不是一个人"的无声证明。

### 2. 给球写信

每周可选。不是写你的感受——是写给你养的心球。

"这周你很安静，我也是。" 存在本地，不发送。写的过程 = 疗愈。

### 3. 球想对你说一句话

喂养满 14 天后解锁。系统根据 14 天情绪轨迹，生成一句回应。

不是"你要坚强"式鸡汤，是 inner-voice 同行者语气："你这周吞了好多沉重的光点。我看你还在亮。"

## 技术可行性

| 引擎 | 现有能力 | 在本产品中的复用 | 改动量 |
|------|---------|----------------|--------|
| inner-voice STEP 0 | 每日趋势感知 | 动态更新情绪卡片库 | 几乎零改动 |
| inner-voice STEP 3 | 悖论→隐喻命名 | 命名建议引擎（简化版） | 去掉深度悖论分析，保留场景化命名 |
| pixel-bloom | Canvas 像素生态系统 | 心球渲染 + 场景切换 | 需新增"积累式生长"状态机 |
| healing-space | WebGL 流体 + 触觉交互 | 拖拽喂养手感 | 需适配移动端触控 |

**最大工作量：** pixel-bloom 的生长算法——情绪向量 → 形态参数 → 渲染。但现有像素生态（植物、光点粒子、毛玻璃 UI）是天然基底。
