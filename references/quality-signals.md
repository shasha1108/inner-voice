# 内容质量信号

> **使用方法：** STEP 8 自检时加载。不是评分标准，不是通过线。是帮 agent 识别"模板生产痕迹"的工具。所有检查方法都是可执行的操作，不依赖主观品味判断。

---

## 5 个结构信号

> 逐一检查。触发 ≥ 2 个 → 内容有模板生产嫌疑 → 回到 STEP 5，优先解决信号指出的问题，而非修补单个指标。

### 信号 1 · 结论可预测性

> **读完标题，能猜到这篇的结论吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 结论里有至少一处，在标题里完全猜不到——不是套路收尾，是真正想通了一个东西 |
| ❌ 触发 | 标题就是结论的压缩版——读完标题就知道结尾会说什么，中间只是填充 |

**检查方法：** 先读标题，猜一个结论。再读结尾。一样 → 触发。

---

### 信号 2 · 中间转折

> **正文中间有没有至少一处思考「拐弯」的地方？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有一处"但我后来发现...""其实不是这样的...""后来才想通..."——不是模板句式，是真的推进了一层 |
| ❌ 触发 | 每段严格对应一个结构功能（描述现象 → 揭示原因 → 给出新视角），没有多余的、意外的内容 |

**检查方法：** 逐段问"这一段如果删掉，前后文逻辑会断吗？"——如果每段都可以删而逻辑不断，说明段落之间没有思考推进，只是按模板排列。

---

### 信号 3 · 细节特异性

> **正文里有至少一个「只有这篇才会出现」的具体细节吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有一个时间/地点/动作/物件的组合过于具体，不像是为了凑场景编出来的 |
| ❌ 触发 | 场景全部是通用模板——换成另一个情绪主题也能直接用 |

**检查方法：** 把情绪主题词换掉（如"焦虑"换成"孤独"），场景还能直接用吗？能 → 触发。

---

### 信号 4 · 语言均匀度

> **全篇句子长度、节奏、精致度高度统一吗？**

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 有粗糙的地方——某句话太短了、太长了、太口语了——但因为它是真实的，读起来不觉得突兀 |
| ❌ 触发 | 全篇像被抛光过——每句话差不多的长度、差不多的精致度、没有一句"出格"的 |

**检查方法：** 找全篇最短的一句和最长的句子。差距 < 3 倍字数 → 语言过于均匀 → 触发。

---

### 信号 5 · 结构要素挤压感

> **正文的每一个段落是否都在"完成一个结构任务"？**

这是门 2（角度类型验证）的副作用——agent 为了满足 Type 结构要素而生产出结构完美的文案，但每个要素之间没有思考推进，只有任务切换。

| 判断 | 标准 |
|------|------|
| ✅ 通过 | 段落之间的过渡是自然推进——"然后我开始想…""后来才意识到…"——下一个想法是从上一个想法里长出来的，不是从检查表里勾出来的 |
| ❌ 触发 | 至少两处段落过渡有"齿轮切换"感——这段在完成"反常识开场"、那段在完成"愤怒转折"、下一段在完成"重新框架"——结构要素的边界清晰得像是不同的人写的 |

**检查方法（两问，任一答"是"→ 触发）：**

1. **功能标签测试**：给每段贴一个结构功能标签（开场钩子 / 痛点展开 / 归因解释 / 重框 / CTA）。有没有两段以上的标签切换让读者感到"上一段还在说 X，怎么突然跳到 Y 了"？有 → 触发。

2. **删段测试**：删掉情绪转折段，前后文是断了吗？还是更顺了？如果删掉后读起来更自然 → 那个转折段是被结构要求逼出来的，不是思考需要 → 触发。

## 隐喻用法一致性检查

> 同一个隐喻在全文中必须用同一种方式使用。不能前半段当作意象/概念用，后半段当作物理空间用。也不能把隐喻用在不匹配的动词上。

**检查方法：**

```
逐句过当前隐喻。每一句里的隐喻是什么用法？
□ 意象/概念？→ "活成了一个谁都可以进来的地方"——"活成"修饰的是状态，"地方"是空间。动词和宾语不匹配 ❌
□ 物理空间？→ "站在那个发光的圆圈里" ✅（圈始终是空间）
→ 两种用法出现在同一篇里 = 隐喻崩溃。动词和宾语不 match = 文字堆砌。

正确做法（选一种，全篇一致。动词和宾语必须匹配）：
✅ "我们活成了一个不会拒绝的人"（"活成" + "人" = match）
✅ "我们把生活变成了一个谁都可以进来的地方"（"生活" + "变成地方" = match）
❌ "我们活成了一个谁都可以进来的地方"（"活成" + "地方" = mismatch）
```

## 情绪基调判断（写作决策工具）

> 在进入门 2 角度类型验证前，先判断这篇的情绪基调属于哪种。这影响如何评估情绪弧线。

| 基调类型 | 定义 | 弧线特征 | 典型主题 |
|---------|------|---------|---------|
| **唤醒型** | 情绪被压抑，需要被看见和释放 | 需要上升弧线：被压住 → 愤怒 → 力量 | 身份撕裂、自我怀疑、愤怒被压抑、倦怠 |
| **陪伴型** | 情绪已被感知，需要被理解和接纳 | 弧线平缓合理：困惑 → 轻轻明白 | 孤独、失望、迷失、被误解、渴望自由 |

**使用方式：** 在 STEP 5 开始写作前判断基调。这影响矛盾句的强度和结尾姿态——唤醒型可以更有力，陪伴型需要更轻。
