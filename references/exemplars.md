# 中文语感参照库

> **使用时机**：STEP 5 文案写作时加载。学的是语言节奏和呼吸方式，不学内容和口吻。
> **核心原则**：中文是"一口气读下来"的语言。好的中文情绪写作靠的是**长短交替的呼吸感**，不是英文式的短句跳跃。

---

## 叙事视角

**默认用"我们"叙事，建立同盟感。**

- "我们"是在说"我和你站在同一边"——读者被邀请进同一个房间，不是被展示一个展品。
- "你"是在说"我在诊断你"——直接把读者放在了被审视的位置。
- "我"是在说"这是我的故事，你可以看看"——自述有力量，但缺少"我们"的同盟感。

大多数时候，"我们"是最好的选择。它让内容从"我懂你"变成"我们一起在这"。

**"你"和"我们"的灵活切换：**

| 人称 | 效果 | 适用场景 |
|------|------|---------|
| **"你"** | 宣言式，有力量感，像在对读者直接说话 | 需要用力量驱动行动时——CTA段、重新框架段、结尾宣告 |
| **"我们"** | 陪伴感，同盟感，像一起在经历 | 开场认同、痛点挖掘、普遍性揭露——让读者感到"不是只有我" |
| **"我"** | 私密感，自述的真实感 | 个人反思、具体场景回忆、配文模式中图里有单一人物 |

**关键：一篇文案可以切换人称，但不能在同一段里混用。** 开场用"我们"建立同盟 → 归因段继续"我们"——读者的信任是连续的。如果突然跳成"你"做诊断，信任就断了。切换发生在段落边界，且是有意的——不是因为忘了用哪种。

**例外情况：**
- 配文模式中图里有明确单一人物 → 可用"我"视角
- 性别相关的主题（如"35+女性的婚姻困境"）→ 可以明确用"我们"的女性视角

## 写作技法（每次写文案时对照）

1. **否定链需要长度来积累重量** — "不是A，也不是B，而是C"——两个否定比一个否定更精准，三个否定比两个更精准。每一次否定都在帮读者排除"这不是我"，到最后一句肯定时，读者已经被精准锚定了。
2. **具体引用 > 抽象概括** — 直接引用"想开点""都会过去的"让人相信"这是真实发生的"。抽象概括"他们敷衍了我"让人感觉"这是编的"。
3. **理解双方视角，不让叙述者变受害者** — 中间插入一句"我知道，他们只是不知道该回应我什么"——从受害者变成能理解双方的人。整篇从控诉变成反思。这也是过审的关键：你不制造对立，你在命名一个共同困境。
4. **先认可对方，再指出问题** — "那些笑容也都是真诚的、温暖的，但是它们太大了"——先给尊严，再说问题。读者更容易接受，平台更不会判定为煽动。
5. **结尾描述现象，让读者自己感受** — 不需要替读者总结"所以你想要的不是答案"。信任读者的理解力。好的结尾是"你看"，不是"所以你应该"。
6. **感官过渡代替声明过渡** — "然后我醒了"是声明。"然后我感觉到了风的吹拂，昏昏沉沉的醒来了"是体验。用身体感觉（风吹/声音/光线变化）来过渡场景，不用"然后""后来""这不是最奇怪的部分"来组织叙事。
7. **"我以为已经忘了...直到某一天"结构** — 先给时间一个跨度（遗忘），再给一个触发（某一天），记忆被激活的瞬间才有重量。比直接说"后来我翻到一张照片"多了一层时间的厚度。
8. **文化引用融入，不显摆** — "万物静默入迷"——知道的人知道出自哪，不知道的人也觉得美。不是掉书袋，是让一句话承载比它字面上更多的含义。

## 参照段落 · 用户原文

> 我对世界做出了反应，
> 但世界给我的反馈，是无力且虚无的。
>
> 我处在一种情感钝化的状态，
> 不是剧烈的忧郁，
> 也不是完全的崩溃，
> 而是一种不好不坏、停滞不前、没有任何动力的灰度状态。

**为什么这段中文好：**
- 第一句是完整的陈述，第二句是对它的否定——不是推翻，是修正。节奏是"我做了A→但得到的是B"。自然的因果。
- "不是...也不是...而是..."——三个否定层层推进，最后落在"灰度状态"。这个递进结构让"灰度状态"四个字有了重量。
- "不好不坏、停滞不前"——中文的四字节奏天然有呼吸感。不需要刻意断句。
- 整段从头到尾一口气读下来，中间没有一处让人"卡住"的地方。

## 参照段落 · 用户原文 2："爱是一个进行时"

> 我们总以为，
> 爱是现在得到了，
> 就可以永远拥有。
>
> 以为确认了恋爱关系，
> 又或者是结了婚，
> 就必须有一个Happy Ending。
>
> 我们以为爱是一个静止的状态，
> 但它是充满摩擦、碰撞、接纳的动态过程。
>
> 爱不仅是十年前他/她对你心动、告白的那个瞬间，
> 而是之后的每一天，
> 是当你们爆发争吵、当你看到他/她不堪、疲惫、脆弱的一面时，
> 你依然选择继续爱他/她。
>
> 爱一个人的过程，
> 是不断去重新认识，
> 并且持续爱上那个不断再变化的他/她。
>
> 爱，是一个进行时。

**为什么这段中文好：**

1. **"我们总以为"≠泛化开场，是邀请反思** — "我们总以为爱是..."是在说一个你和读者共有的误解。不是"我们常常感到忧虑"那种替读者宣布情绪。区别在于：前者邀请读者一起想，后者替读者下判断。

2. **递进式否定，不是矛盾句对仗** — "以为...又或者...以为...但它是..."——这不是结构清单要求的"≥3组矛盾句"。这是在认真想一个问题，然后诚实地给出更复杂的答案。矛盾句应该从思考中自然长出来，不是被规则要求的。

3. **具体场景堆叠** — "爆发争吵、不堪、疲惫、脆弱的一面" ——三个场景串在一起，读者脑海里会自动播放画面。一个场景锚点就够了，三个连在一起会让读者觉得"ta真的经历过这些"。

4. **结尾一个简洁的定义收住** — "爱，是一个进行时。" ——六个字。不多解释。不延伸。不说"所以我们要..."。就是给一个结论，让读者自己消化。结尾的自信来自前面已经论证完了。

5. **没有「」命名，依然好** — 不是每篇都需要命名。这篇讲的是一个重新定义的过程——"爱=进行时"本身就是一个概念的翻新，不需要再包一层隐喻。

**对比我的坏习惯：**
- 如果是我写这篇，我会强行塞一个「爱是动词」或者「进行时的爱」——把已经说清楚的概念再包一层
- 我会在结尾加一句"你上一次重新爱上一个人，是什么时候？"——把信任读者的留白变成逼读者互动的钩子
- 我会把"爆发争吵、不堪、疲惫、脆弱"砍成短句——"你吵了。你看到他最不堪的样子。"——毁了原文的气息

## 参照段落 · 用户原文 3："想开点"（巨大笑脸系列）

> 有一段时间，我试着告诉别人我不太好。
>
> 不是随便说说最近有点累的那种，
> 而是认真的、把我的内心完全敞开的状态。
>
> 我说我好像什么都感觉不到了，
> 不是难过，不是崩溃，也没有剧烈的情绪波动，
> 而是处在一种不好不坏、停滞不前、没有任何动力的灰度状态。
>
> 他们听完了，
> 跟我说："想开点"、"都会过去的"、"你看你多幸福"...
> 然后他们给了我一个大大的笑脸。
> 大到我的视线都被它完全遮住了。
>
> 我也试着回复一个同样的微笑，
> 我知道，他们只是——不知道该回应我什么，
> 而我，应该让他们知道我感觉到被看见了。
>
> 好像这个世界随时准备着很多巨大的笑脸，
> 供所有"不太好"的时刻使用。
>
> 这些话都是真的、正确的，
> 那些笑容也都是真诚的、温暖的，
> 但是它们太大了，
> 大到会把人的痛苦整个盖住，让人看不见痛苦原来长什么样。

**为什么这段中文好——逐技法拆解：**

1. **否定链的长度积累重量**："不是随便说说最近有点累的那种，而是认真的...不是难过，不是崩溃，也没有剧烈的情绪波动，而是处在一种不好不坏、停滞不前、没有任何动力的灰度状态"——两次"不是...而是..."，中间还穿插了一次"也没有"。每一次否定都在画边界，画到第六次否定时，"灰度状态"四个字已经不需要解释了——读者已经知道它是什么。

2. **具体引用制造真实感**："想开点""都会过去的""你看你多幸福"——三个直接引用，你脑子里会有声音。如果换成"他们安慰了我""他们说了一些鼓励的话"——一个字都不信。

3. **中间转折让叙述者从受害者变成理解者**："我也试着回复一个同样的微笑。我知道，他们只是不知道该回应我什么。而我，应该让他们知道我感觉到被看见了。"——这一段是全文的结构转折点。在此之前，"我"是被敷衍的一方；在此之后，"我"理解了对方的无措。整篇文案从控诉变成了反思。这也是过审的关键——你不制造对立，你在命名一个双方都困在里面的共同困境。

4. **先认可再否定**："这些话都是真的、正确的，那些笑容也都是真诚的、温暖的，但是它们太大了"——你先给了这张"笑脸"全部的尊严，然后才说"但是它太大了"。攻击一个已经被你肯定了的东西，比直接攻击更让人信服。读者读到"但是"的时候，心里已经同意了前面的部分。

5. **结尾不替读者总结**："大到会把人的痛苦整个盖住，让人看不见痛苦原来长什么样"——这就是结尾了。没有"所以你想要的不是答案""所以下次如果有人这样安慰你..."——没有。就是让画面留在那里。信任读者的理解力。


## 禁止模仿的特征

以下是我（Claude）写中文时反复出现的坏习惯，每次写文案时必须主动避免：

| 坏习惯 | 示例 | 为什么坏 |
|--------|------|---------|
| 英文短句套中文词汇 | "不是噩梦。不吓人。没有怪物。" / "第三个小时了。你盯着这片蓝色。它没有变深。"——三句之间没有气息连接，是跳的 | 英文的"No X. No Y."排比结构，中文靠逗号连接成一句："不是噩梦，一点也不吓人，那里没有怪物" |
| 感官过渡用声明代替感受 | "然后我醒了。"——这是声明，不是体验 | 好的过渡通过身体感觉完成："然后我感觉到了风的吹拂，昏昏沉沉的醒来了。"——风是肌肤感受到的，醒来是被动发生的 |
| "不是最奇怪的部分。最奇怪的是——" | — | 制造悬念的叙事技巧看起来很聪明，但打断了梦的沉浸感。让你的后半段从"我以为已经忘记了"开始——让读者自己发现这是最奇怪的部分 |
| 结尾总结给答案 | "你只是刚好路过了。"——替读者把感受封装好递过去 | 他们的版本停在"渗进了你的梦里"——画面还在，没有被打包。好结尾是让最后一个画面悬在那里，不是用一句话把它收掉 |
| "就是一"的解释式写作 | "就是一个普通的下午。"——试图用"就是"让普通变诗意 | 去掉"就是"。直接说"那是一个雾蒙蒙的下午"——不需要强调"普通"，让画面自己呈现 |
| 通过在线发现触发 | "我翻到一张照片...就是网上的一张图"——太具体，像情节装置 | 用真实的既视感："我来到一个地方，相似的情景让我又回想起来了"——每个人都经历过这种时刻 |
| 隐喻概念和物理空间混用 | "圈外面的人少了一两个。我们以为这是变小了。圈是为了让我们在里面把背伸直。"——前两句圈是概念（边界），第三句圈是房间（在里面做动作）。读者跟不上 | 全篇选一种用法：圈始终是意象（"圈子小了，人变真了"），或圈始终是物理空间（"站在发光的圆圈里，背伸直了"）。不能混 |
| 伪造SEO数据 | 同安静倦怠=0。声称"长尾搜索词"无任何验证数据 | 没有搜索验证数据=不说。宁可标注"待验证" |
| 动词和宾语不匹配 | "我们活成了一个谁都可以进来的地方"——"活成"修饰的是状态，"地方"是空间。动宾不match | 写完后检查每句的主谓宾。动词和宾语必须是同一类东西。"活成"后面接人或状态，不接地方 |
| 对立修辞冗余填空 | "手是抖的、把背伸直、把气喘匀"——三个身体动作堆在一起。删掉→全文不受影响=它们是多余的 | 每组对立修辞必须通过"删掉测试"。删掉不受影响=去掉。被"需要矛盾句"逼出来的填空=去掉 |
| 「」凭空出现 | 段落中间突然一句「自己画的圈」——无铺垫无解释 | 命名必须有引出。铺垫式（"我们叫它——"）/自然式（"那一道自己画的线。「自己画的圈」"） |
| CTA后面加解释 | "请在评论区留下一个⭕。这代表你给自己画的那个圈。"——读者不需要你解释⭕代表什么 | CTA符号自己会说话。不加解释句。信任读者的理解力 |
| "没有X。没有Y。没有Z。"排比 | "没有深浅。没有远近。没有这里和那里。" | 英文的"No X. No Y. No Z."结构，中文不这么说 |
| 过度使用"——" | "不是的——这种蓝没有维度" | 破折号是英文 em-dash 的替代品，中文靠句子本身的节奏 |
| "你"开头的诊断句 | "你站在哪里，哪里就是正中间" | 替读者下定义=冒犯。用"我"叙述=邀请 |
| 结构检查清单驱动 | 勾完"矛盾句≥3组"就算完成 | 段落之间的逻辑关系不是靠勾结构建立的 |
