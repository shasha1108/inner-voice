## 记录规则

### 入库条件（全部满足）

1. Flow D 复盘确认爆款（数据达标）
2. 用户明确指示提取基因
3. 至少提炼出 1 条可复用基因（不能是"写得好""图好看"）

### 禁止入库

- 数据未达爆款标准
- 只有数据好、说不清为什么好
- Agent 自行判断入库

**Why:** "数据好但不知道为什么好" = 不可复用 = 不是基因。基因是能被下一篇文章直接调用的具体操作。

## 通用视觉公式（来自多个爆款交叉验证）

### ONE MASSIVE 公式

```
单一物理实体（占画面 60%+）
 + 人物极小（仅作比例参照，占画面 ≤20%）
 + 内置冷暖色对比（整体冷调 + 局部暖光点）
= 填满竖图、一眼可读的情绪命名
```

### 视觉主体型 vs 叙事展开型

| 类型 | 特征 | 典型案例 | 结果 |
|------|------|---------|------|
| **视觉主体型** | 一个巨大的单一物理存在主导画面，人物极小，不需要文字就能一眼识别情绪 | 漩涡（被吞没）/ 玻璃盒（隔绝） | 爆款 |
| **叙事展开型** | 需要"理解故事"才能感受到情绪，画面只有"一个人+某个情境" | 地图蓝点消失 / 拆信 / 台词空白 | 必败 |

### 发光透明公式

**正确：深色实体主角 + 旁边/背后的发光透明情绪实体**

- 主角：深色剪影，实体，做日常动作——"身体在场"
- 情绪实体：发光 + 透明 + 人形——"那个感受"，站在旁边/身后/对面
- 色彩：深青蓝大气 + 氰光/白光局部发光
- 构图：主角在画面下方/中央，情绪实体等高或更高，留白在上方

**禁止：** 主角本身透明（鬼魂质感）、整个画面透明物（无对比张力）
