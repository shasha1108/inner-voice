# 失败案例 + 视觉主体性法则

## ⚡ 速查卡（校验步骤只读此节，STEP 3必读全文）

**ONE MASSIVE公式**：单一物理实体(60%+画面) + 人物极小(≤20%) + 内置冷暖色对比 = 停滑型封面

**已验证爆款的共同基因是内在悖论，不是大物体**：
- 漩涡: 挣扎反而加速下沉 ✅3480赞
- 玻璃盒: 看得见外面但出不去 ✅1810赞
- 不会倒的废墟: 碎了但没散 ✅入库

**3问视觉前置测试**（每个候选隐喻必须通过全部3问）：
1. 能压缩成MASSIVE单一物理实体？→ 能=通过；不能=排除
2. 移除人物后，陌生人能感受到情绪？→ 能=通过；不能=排除
3. 在3:4竖图中能占画面60%以上？→ 能=通过；不能=排除

**两大隐喻挖掘陷阱**：
- 字面翻译陷阱：从情绪词→物理形容词→找物体（崩溃→碎→杯子/地面）= 全是同一套损坏皮肤，无悖论
- 缺美感陷阱：ONE MASSIVE通过 ≠ 美；候选必须能说清"为什么这个画面是美的"

> 需要失败案例详细解剖、发光透明正确公式、隐喻独特性自检 → 继续读全文

## 爆款视觉公式（来自实证爆款案例逆推）

```
ONE MASSIVE 单一物理实体（漩涡/玻璃盒/etc.）
 + 人物极小（仅作比例参照，占画面 ≤20%）
 + 内置冷暖色对比（整体冷调 + 局部暖光点）
= 填满一张竖图的、一眼可读的情绪命名
```

| 类型 | 特征 | 典型案例 | 结果 |
|------|------|---------|------|
| ✅ **视觉主体型** | 一个巨大的单一物理存在主导画面，人物极小，不需要文字就能一眼识别情绪 | 漩涡（被情绪吞没）/ 玻璃盒（孤独隔绝） | 爆款 |
| ❌ **叙事展开型** | 需要"理解故事"才能感受到情绪，画面只有"一个人+某个情境" | 地图蓝点消失 / 拆信发现陌生人 / 台词忘词 | 必败 |

## 前置视觉主体性测试（每个候选隐喻进入列表之前必须回答）

```
□ 这个隐喻能压缩成一个MASSIVE的单一物理实体吗？
 → 能（漩涡/盒子/迷宫/墙etc.） = 通过，进入候选
 → 不能（需要展开叙事才能理解）= 直接排除，重想

□ 把人物完全移除，只看这个物理实体本身，陌生人能感受到这个情绪吗？
 → 能 = 通过
 → 不能（没有人物就不知道在说什么）= 排除

□ 在信息流3:4竖图中，这个物理实体能占画面60%以上吗？
 → 能 = 通过
 → 不能（只能是背景或道具）= 排除
```

不过前置测试的隐喻方向，**直接放弃**，不进入候选池。

|------|---------|---------|

| 台词空白（台上忘词） | "有舞台场景" | 需要知道剧场背景+理解"这是20年的角色"，单帧画面只是"一个人站在舞台上" |
| 影子滞后于本体 | "有空间关系" | 依赖观者主动校验物理细节才能识别，普通人不会注意影子的微小偏移 |
| 悬停的硬币 | "有凝固状态" | MJ无法稳定生成悬停硬币；且画面太小，不是主体级别 |

## 失败案例解剖（已验证失败规律 · 创作前必读）

> 以下规律来自对该账号历史爆款（3480❤/1810❤）与失败新作（均≤5❤）的逐帧对比分析。不是理论推测，是实际数据验证的结论。

### 核心死亡陷阱：「形式对了，魂没了」

SOP 化复制了爆款的外壳，但遗漏了爆款的魂：

> **爆款的核心价值 = 一个在那个时刻、对那批用户、第一次被准确说出来的情绪隐喻。**
> 「漩涡」只爆一次，「玻璃盒」只爆一次。用相似的隐喻做相似内容，用户感受到的是「又是这种类型的帖子」。

每次创作的核心问题不是「这个情绪该用什么模板」，而是「这个情绪，有没有一个从没被人这样说过的角度？」

### 失败案例对照表

| 失败类型 | 具体表现 | 根因 |
|---------|---------|------|
| **封面情绪方向错位** | 主题"失控/想停下来"，封面图却是粉色梦幻海边走向地平线（=自由/希望） | 图的情绪信号和标题完全相反，0.5秒被划走 |
| **AI图同质化** | 超现实山野+孤门+人影——这类图已被大量账号复用，用户视觉免疫 | 视觉稀缺性消失，无停滑力 |
| **混入摄影图** | "情绪插画"账号封面用了真实摄影（暗黄雾夜桥上行走） | 破坏"原创插画创作者"的视觉标识 |
| **文案先给概念再解释** | "我们可以把这个状态叫做'自动运转'。不是你变迟钝了，而是……" | 读者感受到被分析而非被陪伴 |
| **标题描述状态而非触发认同** | "最深的困局：随时能出走，却选择留在了原地"（宏大词拉远距离） | 缺乏密语感，像内容介绍而非情绪触发 |
| **把主角做成透明** | `transparent silhouette` / `hollow body` / `see-through figure` 作用于主角 | 鬼魂/泥浆质感 |
| **Magritte-style void cutout** | 「人形镂空透过画布看见背后房间」——连续两套不同方向均得2/10 | MJ精度不足；概念层面根本错误，非Prompt写法问题 |
| **视觉和文案用了不同隐喻** | 文案核心隐喻是「吞回去的不积累」，图片却切换到「镜中无倒影」 | 图文割裂，STEP 6没有从confirmed_metaphor出发 |
| **配色跟随情绪匹配表、忽视停滑力** | 「疲惫/疏离」→「低饱和灰蓝白系」，图片在信息流里隐形 | 情绪匹配≠信息流竞争力；暗调图在视觉嘈杂的信息流里无法被看见 |
| **矛盾句为了格式强行拼凑，牺牲语感** | 「不不」相邻/多重从句叠加 | 只做书面逻辑检验，没有逐句朗读检验 |
| **叙事型隐喻伪装成空间型** | 地图蓝点/拆信/台词空白——需要叙事展开才能理解，单帧无意义 | 没有通过「前置视觉主体性测试」 |
| **隐喻命名"情绪机制"而非"中心感受"** | 「讨好型疲劳」三版全败：命名"如何表演"，没有命名"不知道自己是谁了"这个中心感受 | 读者经验三问只检验体感真实性，不检验是否命中中心感受 |

## ✅ 发光透明视觉元素的正确公式（已验证爆款）

**公式：深色实体主角 + 旁边/背后的发光透明情绪实体**

- **主角**：深色剪影，实体，在做日常动作（坐着/走路/工作）——代表"身体在场"
- **情绪实体**：发光 + 透明 + 另一个人形——代表"那个感受/那个缺席的存在/那个想要的陪伴"，站在主角旁边、身后或正对面
- **色彩**：深青蓝大气 + 氰光/白光局部发光（不是低饱和灰蓝）
- **风格**：`atmospheric digital illustration, painterly style`
- **构图**：主角在画面下方/中央，情绪实体等高或更高，大量留白在上方

**错误版本（同样用了透明元素但必然失败）：**
- ❌ 主角本身是透明的 → 鬼魂/泥浆质感
- ❌ 整个画面的"透明物" → 没有对比张力
- ✅ 主角是实体，旁边/背后才是发光透明 → 对比制造情绪张力，一眼可识别

**示例 Prompt 结构：**
```
atmospheric digital illustration, dark silhouette [doing mundane action] in [blue interior/exterior], a luminous glowing translucent [human presence/figure] [spatial relationship: standing behind/beside/in front], deep teal blue atmosphere, cyan ethereal glow, painterly texture, vast negative space --ar 3:4 --v 8.1 --s 250
```

## 隐喻独特性自检（STEP 3 终选前必做）

在终选输出前，对入选候选隐喻追问：

1. 这个隐喻（漩涡/玻璃盒/etc.）有没有被大量账号用过？如果是，重新发现。
2. 这个隐喻能让第一次看到它的人说"对，就是这个感觉，就是这个形状"吗？
3. 把这个隐喻独立拎出来——它是一个新的「说法」，还是已有说法的变体？

**新说法标准**：用这个比喻形容这个情绪，是第一次（或极少数人说过）。
**变体陷阱**：「困在透明盒子里」≈「困在玻璃盒里」，本质是同一隐喻，不是新发现。
